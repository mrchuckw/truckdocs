Prepare Inspection Mode

Before handing the device to an officer

TruckDocs locks the app screen. The tablet or phone must provide the true device lock.

Before handing the device to an officer, start one of these device-level protections:

  • iPad / iPhone: Guided Access
  • Android: App Pinning / Screen Pinning

This keeps the officer from leaving TruckDocs, opening other apps, opening other browser tabs, or browsing files.

Browser fullscreen is helpful for presentation, but it is not a substitute for Guided Access or App Pinning.



Tap once to arm the button. Tap again within 3 seconds to start locked Inspection Mode. Driver PIN will be required to exit the TruckDocs inspection screen.
